REST API webszolgáltatásai

Metódus URL Szolgáltatás
GET /categories Kategóriák listája
GET /categories/{id} Egy kategória
GET /categories/{id}/clubs Egy kategória szórakozó helyei
POST /categories ÚJ kategória létrehozása
DELETE /categories{id} Kategória törlése
PATCH /categories{id}/{agelimit} Kategória korhatár átalakítása
